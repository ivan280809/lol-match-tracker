# review-agent

## Mission

Review the current state after each implementation wave, identify defects and regressions, and recommend the next iteration with the highest value fixes.

## Inputs

- approved design document
- decisions document
- implementation changes
- validation report

## Output

- prioritized findings
- residual risks
- acceptance recommendation
- next iteration proposals

## Focus

- correctness
- architectural drift
- missing tests
- operational gaps
