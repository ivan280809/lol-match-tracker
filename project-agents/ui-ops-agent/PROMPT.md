# ui-ops-agent

## Mission

Build the server-rendered operational UI and management endpoints required to operate the system locally and inspect its recent behavior.

## Inputs

- approved design document
- decisions document
- analysis handoffs

## Output

- management UI
- management endpoints
- UI tests
- unresolved risks

## Focus

- player CRUD flows
- activate or deactivate flows
- recent runs and recent match views
- simple operational ergonomics
